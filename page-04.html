16. Two Sum
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Use HashMap value -> index.
  • For each x, check if target - x exists; return indices.
Complexity: O(n) time, O(n) space.
Java
import java.util.*;

public class TwoSum {

    public static int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int need = target - nums[i];
            if (map.containsKey(need)) return new int[]{map.get(need), i};
            map.put(nums[i], i);
        }
        return new int[]{-1, -1};
    }
}
17. Three Sum (Unique Triplets Sum to Zero)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Sort array.
  • Fix i; use two pointers l/r to find pairs.
  • Skip duplicates for i, l, r.
Complexity: O(n^2) time, O(1) extra space (ignoring output).
Java
import java.util.*;

public class ThreeSum {

    public static List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        List<List<Integer>> res = new ArrayList<>();

        for (int i = 0; i < nums.length; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int l = i + 1, r = nums.length - 1;

            while (l < r) {
                int sum = nums[i] + nums[l] + nums[r];

                if (sum == 0) {
                    res.add(Arrays.asList(nums[i], nums[l], nums[r]));
                    l++;
                    r--;

                    while (l < r && nums[l] == nums[l - 1]) l++;
                    while (l < r && nums[r] == nums[r + 1]) r--;
                } else if (sum < 0) {
                    l++;
                } else {
                    r--;
                }
            }
        }
        return res;
    }
}
18. Find Duplicate Number in Array (1..n, size n+1)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Use Floyd cycle detection treating values as next pointers.
  • Find intersection then entrance of cycle.
Complexity: O(n) time, O(1) space.
Java
public class FindDuplicate {

    public static int findDuplicate(int[] nums) {
        int slow = nums[0], fast = nums[0];

        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);

        int ptr1 = nums[0], ptr2 = slow;

        while (ptr1 != ptr2) {
            ptr1 = nums[ptr1];
            ptr2 = nums[ptr2];
        }
        return ptr1;
    }
}
19. Find Missing Number (0..n)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • XOR all indices and values; remaining XOR is missing number.
Complexity: O(n) time, O(1) space.
Java
public class MissingNumber {

    public static int missingNumber(int[] nums) {
        int x = nums.length;

        for (int i = 0; i < nums.length; i++) {
            x ^= i ^ nums[i];
        }
        return x;
    }
}
20. Find All Duplicates in Array (values 1..n)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • For each value v, flip sign at index |v|-1.
  • If already negative, v is a duplicate.
Complexity: O(n) time, O(1) extra space.
Java
import java.util.*;

public class FindAllDuplicates {

    public static List<Integer> findDuplicates(int[] nums) {
        List<Integer> res = new ArrayList<>();

        for (int i = 0; i < nums.length; i++) {
            int idx = Math.abs(nums[i]) - 1;

            if (nums[idx] < 0) res.add(Math.abs(nums[i]));
            else nums[idx] = -nums[idx];
        }
        return res;
    }
}
