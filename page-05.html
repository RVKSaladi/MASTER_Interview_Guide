21. Intersection of Two Arrays (Unique)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Put array A in a HashSet.
  • Scan B; if present, add to result set; convert to array.
Complexity: O(n+m) time, O(n) space.
Java
import java.util.*;

public class IntersectionUnique {

    public static int[] intersection(int[] a, int[] b) {
        Set<Integer> set = new HashSet<>();
        for (int x : a) set.add(x);

        Set<Integer> out = new HashSet<>();
        for (int x : b) if (set.contains(x)) out.add(x);

        int[] res = new int[out.size()];
        int i = 0;
        for (int x : out) res[i++] = x;

        return res;
    }
}
22. Union of Two Arrays (Unique)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Add all elements from both arrays to a HashSet.
  • Return set as array/list.
Complexity: O(n+m) time, O(n+m) space.
Java
import java.util.*;

public class UnionUnique {

    public static int[] union(int[] a, int[] b) {
        Set<Integer> set = new HashSet<>();
        for (int x : a) set.add(x);
        for (int x : b) set.add(x);

        int[] res = new int[set.size()];
        int i = 0;
        for (int x : set) res[i++] = x;

        return res;
    }
}
23. Majority Element (> n/2)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Use Boyer-Moore voting.
  • Maintain candidate and count; cancel out different values.
Complexity: O(n) time, O(1) space.
Java
public class MajorityElement {

    public static int majorityElement(int[] nums) {
        int cand = 0, count = 0;

        for (int x : nums) {
            if (count == 0) {
                cand = x;
                count = 1;
            } else if (x == cand) {
                count++;
            } else {
                count--;
            }
        }
        return cand;
    }
}
24. Single Number (every element twice except one)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • XOR all numbers; pairs cancel out.
Complexity: O(n) time, O(1) space.
Java
public class SingleNumber {

    public static int singleNumber(int[] nums) {
        int x = 0;
        for (int v : nums) x ^= v;
        return x;
    }
}
25. Subarray Sum Equals K (count of subarrays)
Arrays + HashMap / Two Pointers / Prefix Sum
Approach
  • Prefix sum + HashMap of prefixSum -> frequency.
  • For each prefix sum ps, add freq(ps-k) to answer; increment freq(ps).
Complexity: O(n) time, O(n) space.
Java
import java.util.*;

public class SubarraySumEqualsK {

    public static int subarraySum(int[] nums, int k) {
        Map<Integer, Integer> freq = new HashMap<>();
        freq.put(0, 1);

        int ps = 0, ans = 0;

        for (int x : nums) {
            ps += x;
            ans += freq.getOrDefault(ps - k, 0);
            freq.put(ps, freq.getOrDefault(ps, 0) + 1);
        }
        return ans;
    }
}
