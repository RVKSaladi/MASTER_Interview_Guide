6. Longest Palindromic Substring
Strings + HashMap / Sliding Window
Approach
  • For each index, expand around center (odd and even).
  • Track best start/end while expanding.
  • Return substring for best range.
Complexity: O(n^2) time, O(1) space.
Java
public class LongestPalindrome {

    public static String longestPalindrome(String s) {
        if (s == null || s.isEmpty()) return "";

        int start = 0, end = 0;

        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);
            int len2 = expand(s, i, i + 1);
            int len = Math.max(len1, len2);

            if (len > end - start + 1) {
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }
        return s.substring(start, end + 1);
    }

    private static int expand(String s, int l, int r) {
        while (l >= 0 && r < s.length() && s.charAt(l) == s.charAt(r)) {
            l--;
            r++;
        }
        return r - l - 1;
    }
}
7. Check if a String Is a Palindrome (Ignore Non-Alphanumerics)
Strings + HashMap / Sliding Window
Approach
  • Two pointers: left and right.
  • Skip non-letter/digit characters.
  • Compare lowercase versions.
Complexity: O(n) time, O(1) space.
Java
public class ValidPalindrome {

    public static boolean isPalindrome(String s) {
        int l = 0, r = s.length() - 1;

        while (l < r) {
            while (l < r && !Character.isLetterOrDigit(s.charAt(l))) l++;
            while (l < r && !Character.isLetterOrDigit(s.charAt(r))) r--;

            if (Character.toLowerCase(s.charAt(l)) != Character.toLowerCase(s.charAt(r))) {
                return false;
            }
            l++;
            r--;
        }
        return true;
    }
}
8. Count Frequency of Characters
Strings + HashMap / Sliding Window
Approach
  • Use int array for ASCII or HashMap for Unicode.
  • Scan string and increment counts.
Complexity: O(n) time, O(1) space for ASCII.
Java
import java.util.*;

public class CharFrequency {

    public static Map<Character, Integer> freq(String s) {
        Map<Character, Integer> map = new HashMap<>();

        for (char c : s.toCharArray()) {
            map.put(c, map.getOrDefault(c, 0) + 1);
        }
        return map;
    }
}
9. Remove Duplicate Characters (Keep First Occurrence)
Strings + HashMap / Sliding Window
Approach
  • Use boolean[256] seen.
  • Append char if not seen; mark seen.
Complexity: O(n) time, O(1) space.
Java
public class RemoveDuplicates {

    public static String dedupe(String s) {
        boolean[] seen = new boolean[256];
        StringBuilder sb = new StringBuilder();

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (!seen[c]) {
                seen[c] = true;
                sb.append(c);
            }
        }
        return sb.toString();
    }
}
10. Find the Most Frequent Character
Strings + HashMap / Sliding Window
Approach
  • Count frequencies.
  • Track max count and its character.
Complexity: O(n) time, O(1) space.
Java
public class MostFrequentChar {

    public static char mostFrequent(String s) {
        int[] cnt = new int[256];
        int best = 0;
        char ans = '_';

        for (char c : s.toCharArray()) {
            int v = ++cnt[c];
            if (v > best) {
                best = v;
                ans = c;
            }
        }
        return ans;
    }
}
