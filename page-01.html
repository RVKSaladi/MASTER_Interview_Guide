1. First Non-Repeating Character
Strings + HashMap / Sliding Window
Approach
  • Count characters while preserving order (LinkedHashMap).
  • Scan the map to find the first character with count == 1.
  • Return placeholder if none.
Complexity: O(n) time, O(1) or O(k) space (k = charset size).
Java
import java.util.*;

public class FirstUniqueChar {

    public static char firstUnique(String s) {
        Map<Character, Integer> freq = new LinkedHashMap<>();

        for (char c : s.toCharArray()) {
            freq.put(c, freq.getOrDefault(c, 0) + 1);
        }

        for (Map.Entry<Character, Integer> e : freq.entrySet()) {
            if (e.getValue() == 1) return e.getKey();
        }
        return '_';
    }
}
2. First Repeating Character
Strings + HashMap / Sliding Window
Approach
  • Use a HashSet to track seen characters.
  • Scan left to right; the first char already in set is the answer.
Complexity: O(n) time, O(k) space.
Java
import java.util.*;

public class FirstRepeatingChar {

    public static char firstRepeating(String s) {
        Set<Character> seen = new HashSet<>();

        for (char c : s.toCharArray()) {
            if (!seen.add(c)) return c;
        }
        return '_';
    }
}
3. Check if Two Strings Are Anagrams
Strings + HashMap / Sliding Window
Approach
  • If lengths differ, return false.
  • Count chars in first string; subtract with second.
  • Verify all counts are zero.
Complexity: O(n) time, O(1) space for ASCII.
Java
public class Anagram {

    public static boolean isAnagram(String a, String b) {
        if (a == null || b == null || a.length() != b.length()) return false;

        int[] cnt = new int[256];

        for (int i = 0; i < a.length(); i++) cnt[a.charAt(i)]++;
        for (int i = 0; i < b.length(); i++) cnt[b.charAt(i)]--;

        for (int x : cnt) if (x != 0) return false;
        return true;
    }
}
4. Find All Anagrams of Pattern in a String
Strings + HashMap / Sliding Window
Approach
  • Sliding window of size p.length().
  • Maintain frequency difference array (pattern counts minus window counts).
  • When all differences are zero, record start index.
Complexity: O(n * A) with naive allZero check (A=256 here).
Java
import java.util.*;

public class FindAllAnagrams {

    public static List<Integer> find(String s, String p) {
        List<Integer> ans = new ArrayList<>();
        if (s == null || p == null || s.length() < p.length()) return ans;

        int[] diff = new int[256];
        for (char c : p.toCharArray()) diff[c]++;

        int m = p.length();

        for (int i = 0; i < s.length(); i++) {
            diff[s.charAt(i)]--;

            if (i >= m) diff[s.charAt(i - m)]++;

            if (i >= m - 1 && allZero(diff)) ans.add(i - m + 1);
        }
        return ans;
    }

    private static boolean allZero(int[] a) {
        for (int x : a) if (x != 0) return false;
        return true;
    }
}
5. Longest Substring Without Repeating Characters
Strings + HashMap / Sliding Window
Approach
  • Use sliding window with HashMap lastSeen index.
  • Expand right; if char repeats inside window, move left.
  • Track maximum window length.
Complexity: O(n) time, O(k) space.
Java
import java.util.*;

public class LongestUniqueSubstring {

    public static int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> last = new HashMap<>();
        int left = 0, best = 0;

        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);

            if (last.containsKey(c)) {
                left = Math.max(left, last.get(c) + 1);
            }

            last.put(c, right);
            best = Math.max(best, right - left + 1);
        }
        return best;
    }
}
