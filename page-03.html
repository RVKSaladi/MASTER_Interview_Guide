11. Find the Second Most Frequent Character
Strings + HashMap / Sliding Window
Approach
  • Count frequencies.
  • Track best and second-best from frequency array.
Complexity: O(n) time, O(1) space.
Java
public class SecondMostFrequentChar {

    public static char secondMostFrequent(String s) {
        int[] cnt = new int[256];
        for (char c : s.toCharArray()) cnt[c]++;

        int first = -1, second = -1;

        for (int i = 0; i < 256; i++) {
            if (cnt[i] == 0) continue;

            if (first == -1 || cnt[i] > cnt[first]) {
                second = first;
                first = i;
            } else if (i != first && (second == -1 || cnt[i] > cnt[second])) {
                second = i;
            }
        }
        return second == -1 ? '_' : (char) second;
    }
}
12. Check if String Characters Can Form a Palindrome
Strings + HashMap / Sliding Window
Approach
  • Count frequencies.
  • At most one character may have an odd count.
Complexity: O(n) time, O(1) space.
Java
public class PalindromePermutation {

    public static boolean canFormPalindrome(String s) {
        int[] cnt = new int[256];
        for (char c : s.toCharArray()) cnt[c]++;

        int odd = 0;
        for (int x : cnt) {
            if ((x & 1) == 1) odd++;
        }
        return odd <= 1;
    }
}
13. Longest Common Prefix
Strings + HashMap / Sliding Window
Approach
  • Take the first string as prefix.
  • Shrink prefix until it matches start of each string.
Complexity: O(total chars) time.
Java
public class LongestCommonPrefix {

    public static String lcp(String[] strs) {
        if (strs == null || strs.length == 0) return "";

        String pref = strs[0];

        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(pref)) {
                if (pref.isEmpty()) return "";
                pref = pref.substring(0, pref.length() - 1);
            }
        }
        return pref;
    }
}
14. String Compression (aabcc -> a2b1c2)
Strings + HashMap / Sliding Window
Approach
  • Scan runs of same character.
  • Append char + run length to StringBuilder.
Complexity: O(n) time, O(n) space.
Java
public class StringCompression {

    public static String compress(String s) {
        if (s == null || s.isEmpty()) return s;

        StringBuilder sb = new StringBuilder();
        int i = 0;

        while (i < s.length()) {
            char c = s.charAt(i);
            int j = i;

            while (j < s.length() && s.charAt(j) == c) j++;
            sb.append(c).append(j - i);
            i = j;
        }
        return sb.toString();
    }
}
15. Reverse Words in a String
Strings + HashMap / Sliding Window
Approach
  • Trim and split by whitespace.
  • Reverse array of words; join with single spaces.
Complexity: O(n) time, O(n) space.
Java
public class ReverseWords {

    public static String reverseWords(String s) {
        String[] words = s.trim().split("\\s+");
        StringBuilder sb = new StringBuilder();

        for (int i = words.length - 1; i >= 0; i--) {
            sb.append(words[i]);
            if (i != 0) sb.append(' ');
        }
        return sb.toString();
    }
}
